Voice Mode Launches for Claude Code — Free Transcription, Real-Time

Anthropic is rolling out voice mode for Claude Code, letting developers issue commands entirely by voice. Hold the spacebar to record, release to transcribe — voice input flows directly into the cursor position in real time alongside normal keyboard use. The transition between voice and typing is seamless. Voice transcription tokens are completely free, with no additional billing. Anthropic engineer Thariq Shihipar confirmed the rollout, noting users activate it with the /voice command.

The feature is currently live for approximately 5% of Claude Code users, with a broader rollout planned over the coming weeks. It's a meaningful quality-of-life upgrade for developers who spend long sessions in the terminal — particularly useful for narrating agentic runs, reviewing diffs out loud, or simply keeping hands off the keyboard during intensive coding sprints.

Automatic Prompt Caching Now Available — No Manual Breakpoints Required
