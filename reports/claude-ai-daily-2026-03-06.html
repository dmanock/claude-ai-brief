Claude Code Gets HTTP Hooks, VSCode MCP Manager & 10 New Voice Languages

A fresh Claude Code release delivers several quality-of-life improvements for developers. HTTP hooks are now supported — Claude can POST JSON to a URL and receive JSON back instead of running a shell command, opening up integrations with external services without subprocess overhead. Native MCP server management is now built into VSCode: use /mcp in the chat panel to enable/disable servers, reconnect, and manage OAuth authentication. The /claude-api skill was added for building applications with the Claude API and Anthropic SDK. Voice STT now supports 10 new languages.
